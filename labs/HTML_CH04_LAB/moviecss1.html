The Princess Bride


If you have not seen The Princess Bride, it is one of those movies that is difficult to describe without spoiling it. The best way I can think to describe it is with a line the grandfather says in response to his grandson wondering if The Princess Bride has any sports in it.

Are you kidding? Fencing, fighting, torture, revenge, giants, monsters, chases, escapes, true love, miracles

Main Cast

  • Cary Elwes: Westley
  • Robin Wright: Buttercup
  • Mandy Patinkin: Inigo Montoya
  • Andre the Giant: Fezzik
  • Wallace Shawn: Vizzini
  • Christopher Guest: Count Rugen
  • Chris Sarandon: Prince Humperdink
